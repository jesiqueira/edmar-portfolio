Visão Geral

O Notebook Manager – Konecta é um sistema interno desenvolvido para gestão, monitoramento e auditoria de notebooks corporativos. Ele foi criado para resolver problemas críticos de governança, rastreabilidade e qualidade de dados em um ambiente com milhares de ativos distribuídos.

Objetivo do Sistema

O sistema centraliza informações de inventário, status operacional, histórico de movimentações, usuários responsáveis e conformidade. Ele substitui planilhas manuais e processos fragmentados, garantindo dados confiáveis para tomada de decisão.

Arquitetura do Projeto

A arquitetura foi projetada para ser robusta, escalável e fácil de manter:

  • FastAPI – API principal, endpoints REST e validações
  • PostgreSQL – Banco relacional com integridade e auditoria
  • SQLAlchemy ORM – Modelagem e persistência
  • Docker – Contêinerização e ambiente padronizado
  • TailwindCSS – Interface moderna e responsiva
  • Jinja2 – Templates dinâmicos para o painel web

Principais Funcionalidades

  • Cadastro e inventário completo de notebooks
  • Histórico de movimentações por usuário, setor e data
  • Dashboard de governança com indicadores de conformidade
  • Monitoramento de status (ativo, manutenção, devolvido, extraviado)
  • Controle de responsabilidade com assinatura digital interna
  • Relatórios exportáveis para auditoria

Modelagem de Dados

O banco foi estruturado para garantir integridade e rastreabilidade:

  • notebooks – dados do equipamento
  • usuarios – responsáveis e colaboradores
  • movimentacoes – histórico completo
  • status_log – auditoria de alterações
  • setores – departamentos e unidades

Fluxo de Operações

  1. Cadastro ou importação de notebooks
  2. Associação a um usuário ou setor
  3. Registro automático de movimentações
  4. Atualização de status e conformidade
  5. Geração de relatórios e auditorias

Desafios e Soluções

1. Dados inconsistentes e duplicados

Solução: validações automáticas + normalização de dados + constraints no banco.

2. Falta de rastreabilidade

Solução: logs completos de movimentação e status.

3. Processos manuais lentos

Solução: painel web com filtros, buscas e relatórios instantâneos.

4. Governança fraca

Solução: dashboard de conformidade com indicadores críticos.

Resultados

  • Redução de erros de inventário
  • Maior controle sobre ativos corporativos
  • Processos mais rápidos e auditáveis
  • Base sólida para expansão futura

Roadmap

  • Integração com Active Directory
  • API para integração com sistemas externos
  • Relatórios avançados com BI
  • Controle de manutenção preventiva
