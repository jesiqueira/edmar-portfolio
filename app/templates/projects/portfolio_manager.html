Visão Geral

O Portfolio Manager – Edmar é o sistema que desenvolvi para gerenciar meu portfólio de forma dinâmica, modular e escalável. Ele organiza projetos, gera páginas automaticamente, aplica animações suaves e mantém uma arquitetura limpa baseada em boas práticas de desenvolvimento.

Arquitetura do Projeto

O projeto segue o princípio SRP (Single Responsibility Principle), garantindo que cada módulo tenha uma responsabilidade única e bem definida. Isso facilita manutenção, testes e evolução do código.

  • routes/ – Rotas separadas por contexto (home, projetos, contato, SEO)
  • templates/ – Componentes reutilizáveis e páginas dinâmicas
  • static/ – CSS, JS e imagens organizados
  • data/ – Estrutura para carregar projetos via JSON
  • services/ – Camada preparada para futuras integrações

Tecnologias Utilizadas

  • Python + Flask
  • Jinja2
  • TailwindCSS
  • HTML/CSS
  • JavaScript
  • Arquitetura Limpa + SRP

Destaques do Sistema

  • Cards animados com stagger
  • Fade-in global nas páginas
  • Templates dinâmicos para cada projeto
  • SEO configurado com Open Graph e Twitter Cards
  • Navbar com efeito de scroll e blur
  • Design responsivo e moderno
